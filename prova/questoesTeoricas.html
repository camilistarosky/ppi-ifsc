Questão 1:

Resposta: Ao digitar um endereço de um site, ao ser enviado e tentar acessar é enviado um diversos dados para que a página seja apresentada corretamente. Após ser encaminhado o site desejado, é encaminhado para o servidor a solicitação, depois é localizado o site solicitado. Após isso, é processado essa solicitação e é enviado para a WEB para ele apresentar e o usuário poder visualizar a página com todas as informações. Por isso, ao acessar cada site tem seu endereço único e correto para acessar diferentes sites com diferentes informações.

Questão 2:

Resposta: O HTML é uma linguagem de marcação de texto, não é uma linguagem de programação. E o CSS é um utilizado para estilização.

Questão 3:

Letra A):

Resposta: O CSS pode ser incluido de três maneiras: 1 - na própria tag HTML, antes de fechar a primeira tag, pode colocar uma tag "style" e definifir o estilo que desejar para essa única tag específica. 2 - no próprio arquivo HTML, dentro do head pode colocar uma tag style onde poderá estilar todos os elementos do body, como todos juntos ou cada um separado através da própria tag, como também pelo ID do elemento ou pela classe. 3 - criar um arquivo CSS e linkar no arquivo HTML. Dessa forma, fica cada um separado e organizado, onde cada arquivo terá o seu objetivo e suas especificações.
